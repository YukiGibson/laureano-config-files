# Laureano's Local Setup

This repository contains my personal configuration files and automation scripts used to set up and manage my local software development environment. It serves as a centralized "source of truth" for my infrastructure, editor settings, and AI-driven development tools.

## Repository Structure

* **`dns/`**: Configurations for my DNS infrastructure using Technitium, configured to filter out spam and advertisements across my local network.
* **`llama_cpp/`**: A collection of shell scripts to automate the lifecycle of models run via `llama.cpp`, including building from source, quantization, and managing specific model versions.
* **`neo_vim/`**: My personal Neovim configuration, built on the `LazyVim` framework.
* **`opencode/`**: Settings and scripts for `opencode`, which enables agentic workflows within my local AI-powered development environment.

## Usage

These files are intended to be used as a reference or to be deployed to their respective locations on my machine (e.g., via symlinks or manual copy) to maintain a consistent and reproducible development environment.
